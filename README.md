# Rivendell
:seedling: A place of peace and beauty where knowledge and stories are preserved.

## Requirements

Install the dependencies using [uv](https://docs.astral.sh/uv/).

```bash
apt install libsdl2-dev libsdl2-mixer-dev libsdl2-ttf-dev # required by snake game
uv install
```
